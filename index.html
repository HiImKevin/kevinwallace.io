Augmented Reality

Mobile AR project, running on Samsung Galaxy S8. See a full writeup on this project here.

The following scene was created using Three.js, an extremely cool/powerful Javascript library that uses WebGL to display 3D graphics natively in the web browser. I started with an array of cubes (5000 of them) and populated them randomly along the x, y, and z axises. Then to make them appear animated/rotating, they were placed in a render loop that's called 60 times every second (aka 60 FPS) to update the scene. The camera can be panned by holding down left click and rotating the view, and the scene can be zoomed in/out on with the middle mouse wheel. This technology can also be used for importing 3D models, adding special effects, playing with materials, shaders, lighting, and so much more.

About

I've been into technology ever since I was little. I started out like most kids, a gamer, and from there began following console modding communities online such as Xbox-Scene and Team Xecuter. I started with building my own PC and soldering modchips into pretty much every console I could get my hands on. Shortly after, I got into programming to better understand the hardware/software exploits I was using, such as one for the orignal Xbox that allowed custom programs and old NES/SNES/Sega emulators to be run on it.
And that was it. Ever since then I've been on this crazy never ending journey of learning as much as possible, not only about tech, but the world we use it in.
